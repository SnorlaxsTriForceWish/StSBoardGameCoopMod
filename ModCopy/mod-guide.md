# ModCopies Directory

## Important Notice

**The files in this directory are READ-ONLY reference copies.**

These are decompiled classes from the JAR files in our dependency list. They are provided for reference purposes only and should not be modified. Any changes needed should be implemented in the main project using patches, hooks, or other modding techniques.

## Directory Structure

### BaseMod
Implements a console and provides hooks to subscribe to or create new console commands.

### ModTheSpire
The core, fundamental modding tool to load external mods for Slay the Spire without modifying base game files.

### SlayTheSpireSource
The source files for the entire Slay the Spire game for reference.

### StSLib
Provides a number of keywords and mechanics for other mods to use. This includes:
- Custom Actions
- Cards
- Keywords
- Relic Hooks
- Power Hooks
- Custom Targeting
- Icons
- Damage Modifiers
- Block Modifiers
- Flavor Text
- Colored Damage Numbers

### TogetherInSpire
A mod that implements simultaneous P2P multiplayer for Slay the Spire.

### Bestiary
A mod that allows the user to see all information about an enemy like their HP ranges, attack patterns, and more.
